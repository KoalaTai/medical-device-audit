# Sample Export Data

This directory contains example exports generated by the assessment tool to demonstrate the output format and quality.

## Files Included

- `sample_export_2024-01-15.zip` - Complete assessment export package
- Contains all three deliverable documents plus JSON results file
- Generated from the sample responses in `../data/sample_answers.csv`

## Export Structure

Each export ZIP contains:

```
deliverables/
├── gap_list.md
├── capa_plan.md  
└── audit_interview_script.md

readiness.json
```

## Usage

These sample files demonstrate:
- Professional document formatting
- Appropriate regulatory language
- Structured recommendations and action plans
- Comprehensive audit preparation guidance

Review these examples to understand the tool's output quality before running your own assessments.
